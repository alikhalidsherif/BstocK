## Font Installation

This project uses the Roboto font. To install the font, please follow these steps:

1. Download the Roboto font from [here](https://www.fontsquirrel.com/fonts/roboto).
2. Unzip the downloaded file.
3. Copy all of the `.ttf` files from the unzipped folder to the `frontend/fonts` directory.
